URLの法則

小説家になろう各ページURL法則について
以下に記す形式のURLはすべてリンクフリーで、許可なくリンクすることができます。

作品ページ

Nコードは作品を管理するためのコードです。

https://ncode.syosetu.com/Nコード/

上記URLにリンクしていただければ適切なURLに誘導します。

R18の場合(ノクターンノベルズ/ムーンライトノベルズ/ミッドナイトノベルズなど)

  • R18作品に非R18用URLで接続した場合、novel18.syosetu.comに自動転送します
  • 非R18作品にR18用URLで接続した場合、ncode.syosetu.comに自動転送します
レスポンス

SEOの観点からnovel18.とncode.間の移動はHTTPレスポンスヘッダで 301 Moved Permanentlyを返します。 それ以外の場合は永続的ではなく、一時的な移動なので302を返します。

端末判別

従来はnetUserAgentMobileを使用していましたが、メモリ使用量などの面から軽量ライブラリを独自開発しました。

マイページ

ユーザIDは数値です。ユーザごとに発行されています。

https://mypage.syosetu.com/ユーザID/
