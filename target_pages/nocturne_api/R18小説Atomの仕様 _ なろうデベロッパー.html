R18小説Atom

Atom(アトム)フィードで作品更新情報を提供しています。Atomを使うことで作品の更新情報をすばやく把握することができます。

このページで取り扱う情報には、18歳未満が取り扱うのにふさわしくない情報が含まれています。

AtomフィードのURL

種類別にAtomフィードを発行しています。

https://api.syosetu.com/noc_allnovel.Atom

ノクターンノベルズに投稿された男性向けR18新着作品のAtomフィードです。

https://api.syosetu.com/mnlt_allnovel.Atom

ムーンライトノベルズに投稿された女性向けR18新着作品のAtomフィードです。

https://api.syosetu.com/mnltbl_allnovel.Atom

ムーンライトノベルズに投稿された女性向けR18新着作品[BL限定]のAtomフィードです。

https://api.syosetu.com/mnltnomal_allnovel.Atom

ムーンライトノベルズに投稿された女性向けR18新着作品[BL除く]のAtomフィードです。

https://api.syosetu.com/mid_allnovel.Atom

ミッドナイトノベルズに投稿された大人向けR18新着作品のAtomフィードです。

Xユーザ別Atom

https://api.syosetu.com/writernovel/○○○○.Atom

Xユーザ単位の作品更新情報です。
○○○○のところにXIDが入ります。なお、Xユーザ名は敬称略としています。

Atomフィードで提供される情報

エピソード単位で情報を提供します。リンク先もエピソード単位となっています。
<summary>には作品本文出だし100文字が入っています。
<title>には連載作品[○○○○-■■■■](エピソード△)]のように作品情報が記載されます。
○○○○には作品名、■■■■はエピソードタイトル、△にはエピソード番号が入ります。

Atomフィードの利用方法

AtomフィードはAtom対応ウェブブラウザ(InternetExplorer7以上やFirefoxなど)で読み込むことにより誰でも簡単に利用できます。
また、PHPやPerl、Javaなどのプログラミング言語からもライブラリを使うことで読み込みできます。
Atomフィードで提供される情報の利用はご自由にどうぞ。

なろう小説APIやなろうR18小説API(R18)と併用すると高度な小説サイトが作成できます。

遅延と最大件数

Atomフィードは平均10分ごとの更新です。Atomフィード作成時に瞬時に最寄の小説データーベースに新着作品情報を問い合わせAtomフィードが作成されます。
しかし、マスターデータベースとスレーブデータベースに何らかのネットワーク障害が発生した場合に限り、数時間情報が遅れていることがあります。予めご了承ください。

Atomフィードには通常、新着100エピソードが掲載されています。連載作品で連続次話投稿した場合、同一作品が同一Atomフィードに複数表示されていることがあります。

負荷削減のため、allnovel.Atomは過去24時間、noc_allnovel.Atomは過去168時間、mnlt_allnovel.Atomは過去240時間、その他は過去360時間のみ取得します。取得範囲に最大表示件数である100作品が掲載されなかった場合に限り、Atomフィードに掲載される作品が100以下の可能性があります。

利用制限

基本的に利用制限はありませんが、小説家になろうグループのほかのページと同様で、一人の利用者から連続した接続を確認した場合、利用者のIPアドレス単位で一時的にアクセスを制限することがあります。
