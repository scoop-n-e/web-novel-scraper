WWWCメタタグチェックツール対応に関して

WWWCとは？

WWWCを使うことでウェブサイトの更新をチェックできます。
以下、WWWCサイト上より引用

WWWCはWindowsで動作するWebページの更新チェッカです。
よく見るWebページのURLを登録しておいて更新のチェックを行い、更新されたWebページを通知してくれるというものです。
WWWCに登録してあるURLを最大30個同時にチェックを行います。自分でブラウザを使用して更新されてないか見に行くより遥かに高速に更新のチェックを行うことができます。
基本的な使い方はWindowsに付属しているエクスプローラ(ファイルを操作するソフト)と同じ使い方です。
WWWCはフリーソフトウェアです。配布、転載は自由です、特に許可を取る必要はありません。

小説家になろうグループのWWWC対応状況

作品ページ(例：https://ncode.syosetu.com/n0000a/)はWWWCなどのチェックツールに対応しています。
作品の更新をブラウザの起動なしで素早く知りたい場合などにご利用下さい。

<meta>タグの仕様

WWWCに更新日を伝えるために、作品ページには<head>～</head>タグ内に以下のような<meta>タグを埋め込んでいます。

<meta name="WWWC" content="2009/07/21 00:00">

日付は作品の最終掲載日です。日付形式は西暦/月/日 時:分の形式です。
時刻が更新されるのは作品投稿と次話投稿時のみです。作品編集は時刻反映されません。

WWWC接続時の文字コード

作品ページはWWWCでアクセスされた場合のみShift_JISで送信します。
小説家になろうの文字コードは原則としてUTF-8となっております。しかし、WWWCはUTF-8を読むことができず、文字化けしてしまいます。
この問題を解決するため、UserAgentにWWWCの文字が入っている場合のみ、Shift_JISに変換して送信しております。
UTF-8→Shift_JISへの強制変換です。一部の文字が文字化けする場合があります。予めご了承ください。

WWWC以外で作品の更新を把握する手段

小説家になろうグループでは作者単位のAtomも配信しています。更新チェックにご利用ください。
