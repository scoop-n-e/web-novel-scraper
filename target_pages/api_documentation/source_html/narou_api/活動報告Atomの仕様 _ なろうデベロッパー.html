活動報告Atom

Atom(アトム)フィードで活動報告の更新情報を提供しています。Atomを使うことで新着活動報告をすばやく把握することができます。

AtomフィードのURL

https://api.syosetu.com/blogatom.Atom

小説家になろう全体の活動報告のAtomフィードです。

ユーザ別Atom

https://api.syosetu.com/writerblog/○○○○.Atom

ユーザ単位の活動報告です。○○○○のところにユーザIDが数字で入ります。
なお、ユーザ名は敬称略としています。

Atomフィードで提供される情報

  • <summary>
    活動報告の出だし100文字が入っています。
  • <title>
    活動報告のタイトルが記載されます。
  • <comment>
    活動報告を書いた人のユーザIDとユーザ名が ユーザID/ユーザ名 のように/(スラッシュ)区切りで記載しています。

Atomフィードの利用方法

AtomフィードはAtom対応ウェブブラウザで読み込むことにより誰でも簡単に利用できます。
また、PHPやPerl、Javaなどのプログラミング言語からもライブラリを使うことで読み込みできます。
Atomフィードで提供される情報の利用はご自由にどうぞ。

遅延と最大件数

Atomフィードは平均10分ごとの更新です。Atomフィード作成時に瞬時に最寄の活動報告データーベースに新着活動報告情報を問い合わせAtomフィードが作成されます。
しかし、マスターデータベースとスレーブデータベースに何らかのネットワーク障害が発生した場合に限り、数時間情報が遅れていることがあります。予めご了承ください。

Atomフィードには新着100件が掲載されています。

利用制限

基本的に利用制限はありませんが、小説家になろうグループのほかのページと同様で、一人の利用者から連続した接続を確認した場合、利用者のIPアドレス単位で一時的にアクセス規制することがあります。
